# Simple HTML Application

This is a basic HTML application with CSS and JavaScript that can be deployed using CI/CD.

## Features

- Simple webpage with a button that changes text on click
- Responsive design
- Automated deployment to GitHub Pages via GitHub Actions

## Local Development

1. Clone the repository
2. Open `index.html` in your browser

## Deployment

The application is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.

To enable GitHub Pages:
1. Go to your repository settings
2. Scroll to "Pages" section
3. Select "GitHub Actions" as the source

## Usage

- Click the "Click Me!" button to see the message change.

## Troubleshooting

- If the page doesn't load, ensure all files are in the correct paths.
- For deployment issues, check the Actions tab in GitHub for logs.
